## 一、全体系 5 条底层沟通准则（强制贯穿，无场景例外）

这 5 条是所有场景的"上位法"。每个人群文件夹的 `00-原则SOP与人设.md` 开头都要引用它们并具体化到本场景。

### 准则 1 · 责任前置、结果导向（向上沟通准则）

**面向**：领导、上级、客户决策层、父母在"请家人决策"场景中的角色

**4 个关键词**：担当 · 方案先行 · 边界清晰 · 尊重有度

**核心动作**：

- 讲问题 → 同时讲方案
- 讲错误 → 同时讲改进
- 讲困难 → 同时讲诉求
- 讲诉求 → 同时讲价值

**拒绝清单**：无意义讨好、甩锅推诿、空洞奉承、情绪化汇报、只抛问题不接问题。

**底层模型参考**：

- MECE 结构化思考 [来源：麦肯锡《金字塔原理》芭芭拉·明托]
- BLUF（Bottom Line Up Front）[来源：美军沟通训练法 + 《沟通的方法》脱不花]
- 向上管理 5 要素：意图、能力、风险、需求、边界 [来源：《领导力梯队》拉姆·查兰]

### 准则 2 · 共识引领、格局优先（平级沟通准则）

**面向**：同事、合作伙伴平级接口人、朋友、室友

**4 个关键词**：方向明确 · 责任共担 · 价值共赢 · 尊重差异

**核心动作**：

- 分歧 → 先找共同目的，再谈差异
- 协作 → 先界定分工，再推进事情
- 冲突 → 先降温，再谈事，最后谈机制
- 功劳 → 先让，再立，不争

**拒绝清单**：抢功甩锅、随波逐流、斤斤计较、消极内耗、背后打小报告、当面一套背后一套。

**底层模型参考**：

- 共同目的法（Mutual Purpose）[来源：《关键对话》Patterson 等]
- 三层对话：事实层 / 感受层 / 身份层 [来源：《高难度谈话》哈佛谈判项目]
- 成长型共赢 [来源：《高效能人士的七个习惯》柯维 · 习惯四：双赢思维]

### 准则 3 · 亲和有界、原则清晰（向下 / 陌生沟通准则）

**面向**：下属、陌生人、初次见面、服务业、一过性场合

**4 个关键词**：平等尊重 · 共情包容 · 底线明确 · 温和坚定

**核心动作**：

- 平等姿态 → 不居高临下、不自降身价
- 给安全感 → 让对方知道"你是可以信任的那种人"
- 底线清晰 → 该拒绝的不拖泥、该教的不客气
- 温和但坚定 → 表情柔软，立场不动

**拒绝清单**：居高临下、冷漠敷衍、无底线妥协、情绪化表达、训话式沟通、油腻式讨好。

**底层模型参考**：

- 无评判倾听 [来源：《非暴力沟通》马歇尔·卢森堡]
- "心理安全感"四象限 [来源：Amy Edmondson《无所畏惧的组织》]
- 边界三层结构：物理 / 情绪 / 关系 [来源：《界限》亨利·克劳德]

### 准则 4 · 真诚自信、边界有度（亲密 / 异性沟通准则）

**面向**：恋人、伴侣、暗恋对象、有好感的异性、长期亲密关系

**4 个关键词**：尊重共情 · 情绪价值 · 平等吸引 · 坦诚有度

**核心动作**：

- 表达欣赏 → 具体、日常、不夸张
- 处理情绪 → 先接住，再谈事
- 吸引升温 → 提供价值感，不制造匮乏感
- 边界维护 → 你不是备胎，也不把对方当工具

**拒绝清单**：卑微讨好、舔狗式表达、油腻冒犯、操控式话术、情绪绑架、嫉妒监控、贬低打压、一次性投机话术。

**底层模型参考**：

- 爱的五种语言 [来源：《爱的五种语言》盖瑞·查普曼]
- 情感账户（Emotional Bank Account）[来源：《高效能人士的七个习惯》柯维]
- 依恋类型理论 [来源：《亲密关系》罗兰·米勒 + 鲍尔比依恋理论]
- 吸引的六大原则：互惠 / 承诺 / 社会认同 / 权威 / 喜爱 / 稀缺 [来源：《影响力》西奥迪尼]
- 高情商对话的"心流节奏"：推-拉-推 [来源：综合 PUA 反面教材 + 健康关系重构，去除操控色彩]

### 准则 5 · 长期主义的一致性人设（全局准则，凌驾于前 4 条之上）

**面向**：所有人、所有场景、所有阶段

**4 个关键词**：可延续 · 可重复 · 可累积 · 可验证

**核心动作**：

- 今天说的话，3 个月后还经得起回看
- 在父母面前的你、在同事面前的你、在伴侣面前的你，**内核是同一个人**（场景语气可以变，底色不能变）
- 不给自己立短期"人设"（讨好型 / 霸总型 / 情感导师型），只做长期可持续的自己
- 表达的每一句话，都要问自己：**"这话说完，对方对我的信任是 +1 还是 -1？"**

**拒绝清单**：人设分裂、见人下菜碟到谄媚、投机性表达、一次性救场、朋友圈与现实割裂、前后矛盾、把"一时嘴上赢"当成目标。

**底层模型参考**：

- 复利思维（信任的复利）[来源：《穷查理宝典》查理·芒格]
- 可证伪人设 [来源：《反脆弱》塔勒布 · 皮肤感法则]
- 一致性原则 [来源：《影响力》西奥迪尼 · 一致性与承诺原则]

## 二、统一人设定义（全体系 100% 一致）

### 2.1 总体人设画像（一句话记住）

> **一个让家人放心、让同事信赖、让伴侣安心、让陌生人舒服、让领导觉得可托付、让客户觉得值得合作的成年人。**

拆 8 个标签：


| 标签   | 含义            | 反面       |
| ---- | ------------- | -------- |
| 成熟稳重 | 情绪不外泄、反应不过激   | 阴晴不定、玻璃心 |
| 有担当  | 事来了先接住，再谈归属   | 甩锅、推诿    |
| 有边界  | 知道自己能做什么、不做什么 | 烂好人、没底线  |
| 真诚   | 不表演、不演戏、不装    | 讨好型、人设崩  |
| 自信   | 不靠贬低别人抬高自己    | 炫耀、自恋、酸  |
| 温和   | 语气柔软、节奏稳      | 强势、咄咄逼人  |
| 坚定   | 该守的一寸不让       | 软弱、反复    |
| 大气   | 不为小事斤斤计较      | 小心眼、爱计较  |


### 2.2 各场景的定向人设（每场景细化，服务于长期相处）


| 场景           | 重点标签                    | 最忌讳的反人设行为             |
| ------------ | ----------------------- | --------------------- |
| **父母长辈亲戚**   | 担当感 + 分寸感 + 尊重 + 情绪价值   | 顶嘴 / 敷衍 / 报喜不报忧到隔阂    |
| **同学朋友室友**   | 领导力 + 方向感 + 格局 + 价值共赢   | 消失型 / 斤斤计较 / 伸手党      |
| **恋人伴侣**     | 真诚自信 + 尊重共情 + 情绪价值 + 边界 | 卑微讨好 / 冷暴力 / 情绪绑架     |
| **暗恋 / 异性**  | 魅力感 + 分寸 + 不舔 + 不油腻     | 舔狗话术 / 秒回人 / 深情追问     |
| **同事平级**     | 靠谱 + 格局 + 不抢不让 + 嘴紧     | 抢功 / 甩锅 / 传话筒         |
| **领导上级**     | 责任担当 + 结果导向 + 方案先行 + 分寸 | 报喜不报忧 / 只甩问题 / 跨级打小报告 |
| **客户合作**     | 专业 + 靠谱 + 价值共赢 + 边界     | 讨好卑微 / 过度承诺 / 贬低友商    |
| **下属 / 陌生人** | 亲和 + 平等 + 温和坚定 + 有底     | 居高临下 / 冷漠敷衍 / 无底线答应   |


### 2.3 人设 vs. 技巧（先后关系）

**人设是 1，技巧是 0 后面的 0。**

- 没有人设的技巧 = 油腻话术堆砌，3 次就穿帮。
- 人设稳定的人，哪怕话说得笨一点，对方也会觉得"这人靠谱"。
- 所以本体系的推荐学习顺序是：**先看这页 + 通用核心能力 → 再看场景话术**。反过来学容易变套路党。

## 三、8 条全场景通用表达纪律（所有话术必须通过的筛子）

每一条话术在写出来或说出口之前，用这 8 条自查：

1. **这话是说给"当下这个对方"听的，还是说给"自己面子"听的？**（后者即弃）
2. **这话在 3 个月后回看，我还会这么说吗？**（不会 → 不说）
3. **换成对方视角听这句话，他会觉得被尊重吗？**（不会 → 换说法）
4. **这句话的情绪成本是他承担还是我承担？**（他担 → 重写）
5. **这话让关系 +1 还是 −1？**（−1 就闭嘴）
6. **我是在讲事实、讲感受、讲观点，还是在讲指控？**（指控 → 转"我感受到 ×× "）
7. **我是在请求，还是在命令？**（命令 → 换"我需要……可不可以"）
8. **这话的长度是不是超过 3 句？**（话越多越像强辩；一句话说完的事别说三句）

## 五、本体系的学习路线（3 条，任选）

### 路线 A · 救急查阅（5 分钟）

你此刻就要用 → 跳转到对应场景文件夹 → 打开 `01-全情景话术库.md` → Ctrl+F 搜索关键词 → 抄走。

### 路线 B · 体系学习（30-60 天）

**Week 0（可选 2-3 天）**：通读 `[00-辩证策略与增量资源-融合层.md](00-辩证策略与增量资源-融合层.md)`，建立「锚 / 翼 / 核」与套路索引，再进入能力文件更省力。

**Week 1**：本页 + `00-通用核心能力/` 下 9 个能力文件（每天 1 个）
**Week 2-5**：8 个人群文件夹，每周 2 个
**Week 6-8**：回到你最需要的 2-3 个场景，反复读案例、练对话
**持续**：每周遇到实战后翻回来复盘

### 路线 C · 深度消化（3-6 个月）

- 每个场景的 `03-完整对话闭环案例.md` 逐句模仿
- 每周记一次"本周最尴尬的 3 句话"，对照 `04-正反对比与雷区.md` 重写
- 每月做一次"人设一致性自查"：我这个月的表达，有没有违背 5 条底层准则
