## 二、辩证式体系 → v2 调用地图（思路整合）

### 2.1 四条宪法 ↔ v2 五准则（叠加关系）


| 辩证式宪法                 | 与 v2 的叠读方式                                          |
| --------------------- | --------------------------------------------------- |
| **立场锚定**：核心诉求与底线不动摇   | 对齐 v2 **准则 1—4** 中各场景的「原则 / 红线」；只调表达方式，不调底线         |
| **辩证表达**：选择性真诚 + 梯度披露 | 对齐 **准则 4「坦诚有度」** + 下文 T0—T5；**事实不造假**与 v2「来源不伪造」同阶 |
| **迂回落地**：每步指向目标，不为绕而绕 | 对齐 **准则 5 长期一致**：迂回不能变成「油、赖、人设分裂」                   |
| **人设统一**：有担当、不卑微、不油滑  | 与 v2 **统一人设**同一句；策略是「翼」，人设是「骨」                      |


### 2.2 五个可执行模型（压缩版，完整定义见辩证式原文）

1. **真诚度梯度 T0—T5**：风险高 → T 低；信任高 → T 高；**事实类须真，底牌不交**。[来源：辩证式策略沟通体系 · 模块一]
2. **立场—表达分离**：锚（不可交易）与路（入口 / 披露度 / 节奏 / 姿态）分开设计。[来源：同上]
3. **攻守转换**：边界句 → 换战场到条件与选项 → 小让步换范围 / 标准 / 资源。[来源：同上]
4. **信息分层 I1 / I2 / I3**：用 I2 换对方关键信息，I3 始终可回收。[来源：同上]
5. **关系—目标平衡**：保温句 + 立场句 + 下一步句；避免「赢了对话、输了下次」。[来源：同上]

## 四、推荐「调用」工作流（把三份材料合成一套用法）

1. **平日常规**：v2 `00-底层` → `00-通用核心能力` → 具体人群 `01` 话术。
2. **要高博弈、但不想人设崩**：打开 **辩证式原文**，先写「锚句」再选套路（第二节映射表）。

4. **要大段话术范例**：`高情商社交话术_增量融合版` 全文检索关键词（与 v2 场景文件夹互补，注意**口语与人设**仍以 v2 为准）。
