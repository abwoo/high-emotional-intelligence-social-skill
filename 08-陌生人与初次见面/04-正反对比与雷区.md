## 二、应急沟通预案（6 种）

### 1. 对方醉酒 / 情绪失控靠近你

- 不争论；保持距离；求助工作人员或报警（必要时）。
- 话术：「不好意思我先离开。」

### 2. 密闭空间（电梯 / 网约车）感到不适

- 提前下车 / 到有人楼层出电梯；分享行程给亲友。
- 不与司机争执路怒。

### 3. 被道德绑架（「帮帮我」「就这一次」）

- 重复拒绝 + 离开；不掏现金给陌生人「应急」（防诈骗）。

### 4. 被拍照 / 被录视频

- 明确「请不要拍我」；涉及侵权可要求删除并求助场所管理方。

### 5. 冲突升级（踩脚、插队）

- 先道歉或简短说明事实；对方辱骂 → 不跟骂，找第三方（乘务、保安）。

### 6. 孩子 / 老人走散求助

- 帮找工作人员或报警；不单独带孩子离开公共场所。

## 三、绝对雷区（12 条）

| 雷区 | 长期 / 即时影响 | 补救 |
|---|---|---|
| 1. 跟到卫生间 / 住所外蹲守 | 违法风险、对方恐惧 | 立即停止，永不再犯 |
| 2. 死缠烂打要微信 | 被拉黑、报警 | 一次拒绝即停 |
| 3. 肢体越界 | 冲突、治安事件 | 道歉 + 后退 + 不再接触 |
| 4. 开黄腔、物化对方 | 社交性死亡 | 郑重道歉，不辩解 |
| 5. 打听收入、婚恋、住址 | 信任归零 | 不再问，转移公开话题 |
| 6. 公开贬低对方职业 / 地域 | 冲突 | 当场道歉 |
| 7. 利用同情心借钱 | 诈骗嫌疑 | 一律引导找警察 / 官方救助 |
| 8. 偷拍、传播隐私 | 法律风险 | 删除、道歉、承担后果 |
| 9. 酒桌劝酒给陌生人 | 意外责任 | 不劝酒，劝停 |
| 10. 为「练胆」骚扰式搭讪 | 恶习固化 | 改场合、改方式（活动内正当社交） |
| 11. 把拒绝当「欲擒故纵」 | 骚扰 | 尊重字面意思 |
| 12. 带陌生人去偏僻处 | 安全风险 | 只去公开、明亮、有他人的地方 |

---

## 四、已冒犯对方 · 3 步补救

1. **立刻停**：不再靠近、不再发消息。
2. **短道歉**：「刚才 ×× 是我不对，抱歉。」不长篇自我辩护。
3. **用行动证明**：以后保持距离；必要时通过共同场合中性相处。

---

## 五、一句总结

> 陌生人的好感来自「舒服与安全」，不是来自「话多和热情」。
> **短、真、有礼、有界**——四条做到，你就已经超过大多数临场发挥。
